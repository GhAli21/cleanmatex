# Documentation Archive Migration Skill

This skill retires legacy or duplicate docs safely.

Use it when:

- old docs should stop acting like active truth
- you need redirect stubs to a canonical replacement
- historical docs should be prepared for archive or support-only status

Its goal is preservation plus clarity, not silent deletion.

Example trigger phrases:

- "archive these old docs safely"
- "turn this into a redirect stub"
- "retire this legacy documentation"
