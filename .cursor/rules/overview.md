# Project Overview Rules

## Overview
Core project information and differentiators for CleanMateX platform.

## Rules

### Project Identity
- CleanMateX is a multi-tenant SaaS platform for laundry and dry cleaning management
- Target market: GCC region (Saudi Arabia, UAE, Oman, Kuwait, Bahrain, Qatar)
- Default currency: OMR (Omani Rial)
- Default timezone: Asia/Muscat
- Supported languages: English and Arabic (RTL mandatory)

### Core Value Proposition
- Per-piece tracking of laundry items
- Quality assurance gates before order completion
- Digital-first customer engagement
- Multi-branch support per tenant
- Real-time order tracking

### Key Differentiators
- Full bilingual support (English/Arabic) with RTL
- Multi-branch operations per tenant
- Mobile-first approach (customer, driver, store apps)
- GCC payment gateway integration (HyperPay, PayTabs, Stripe)
- WhatsApp Business API integration
- Enterprise security with Row-Level Security (RLS)

### Business Model
- SaaS subscription with tiered pricing (Free, Starter, Growth, Pro, Enterprise)
- Optional transaction fees for SMS/WhatsApp
- Future marketplace commission revenue
- White-label options for enterprise clients

### Development Philosophy
- Built for solo developer with AI assistance
- Prioritize free/open-source tools
- Progressive enhancement approach
- Revenue-generating features first
