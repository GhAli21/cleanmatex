# Documentation Map Rules

## Overview
Rules for locating project documentation files.

## Rules

### Documentation Locations
- Requirements specifications: `docs/Requirments Specifications/clean_mate_x_unified_requirements_pack_v_0.12.1.md`
- Implementation plans: `docs/plan/` (master plan and feature-specific plans)
- Database design: `docs/Database_Design/`
- Configuration files: `docs/config/` (e.g., `supabase_keys.md`)

## Conventions
- All documentation must be placed under `docs/` directory
- Follow documentation structure rules defined in `documentationrules.md`
- Maintain lookup files for discoverability
