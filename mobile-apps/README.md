# CleanMateX Mobile Apps

This directory is reserved for Flutter-based mobile applications in the CleanMateX platform.

## Intended Scope

Planned app surfaces include:

- customer application
- driver application
- store or staff mobile experience

## Current Repository State

At the time of this documentation refresh, this area does not yet have a mature local documentation set or visible app-specific markdown guides. Treat the mobile roadmap as planned scope unless a concrete implementation exists in this directory.

## Guidance

- use `docs/plan/master_plan_cc_01.md` and relevant feature docs for roadmap context
- use `mobile-apps/.clauderc` as implementation guidance only, not as proof that a mobile module is already fully implemented
- when app code and folders are added, create colocated app READMEs and feature/module docs that reflect real implementation status
