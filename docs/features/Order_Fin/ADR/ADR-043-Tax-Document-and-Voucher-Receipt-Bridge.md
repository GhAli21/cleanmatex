# Voucher Receipt Is Temporary Tax Bridge Only

**Status:** Accepted  
**Area:** Tax Document / Receipt  
**Date:** 2026-05-30  
**Decision Type:** Architecture Decision Record

## Context

Until Tax Document Module is ready, voucher receipt may need customer-facing tax details.

## Decision

Voucher receipt can be hardened as a temporary bridge, but it is not the long-term tax document system.

## Consequences / Implementation Rule

Long-term compliance requires Tax Document Module for fiscal/e-invoicing behavior.
