# Customer Category Categorization - User Guide (Tenant App)

## Catalog – Customer Categories

1. Go to **Catalog & Pricing** → **Customer Categories**
2. View tenant categories (system + custom)
3. **Create:** Add custom categories (e.g. HOTEL, B2B_CORP)
4. **Edit:** System categories – only Name and Display Order editable
5. **Delete:** System categories cannot be deleted

## Creating Customers

**B2B Customers:**
- Select a **B2B Category** from the dropdown (required when B2B categories exist)
- Only B2B categories are shown

**Guest / Stub Customers:**
- Optionally select a **Customer Category**
- Categories shown are individual (non-B2B) only
