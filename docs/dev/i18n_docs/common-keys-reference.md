# Common Keys Reference

List of some examples of common keys used across the application. Use `tCommon('key')` with `useTranslations('common')`.

## Actions

| Key | EN | Usage |
|-----|----|-------|
| save | Save | Form submit, save button |
| cancel | Cancel | Cancel button, modal close |
| delete | Delete | Delete action |
| edit | Edit | Edit action |
| create | Create | Create new entity |
| update | Update | Update action |
| submit | Submit | Form submit |
| reset | Reset | Reset form |
| clear | Clear | Clear selection |
| clearFilters | Clear Filters | Clear filter button |
| close | Close | Close modal/dialog |
| back | Back | Back navigation |
| next | Next | Pagination, wizard |
| previous | Previous | Pagination |
| actions | Actions | Table column, menu |
| view | View | View action |
| download | Download | Download action |
| upload | Upload | Upload action |
| refresh | Refresh | Refresh data |
| print | Print | Print action |

## States

| Key | EN | Usage |
|-----|----|-------|
| loading | Loading... | Loading state |
| saving | Saving... | Save in progress |
| deleting | Deleting... | Delete in progress |
| error | Error | Error message |
| success | Success | Success message |
