# CleanMateX Prompt System

This folder contains the reusable prompt system for CleanMateX.

Use it when you want:

- the right prompt for planning, implementation, review, UI/UX, data/API, or strategy work
- reusable quality blocks and add-on lines
- guidance on how to create better prompts
- examples of finished prompts
- instructions for using the `build-prompt` skill

## Notes

- Prompts do not replace repository rules.
- Prompts do not replace domain skills such as `/frontend`, `/backend`, `/database`, `/i18n`, `/multitenancy`, `/implementation`, or `/documentation`.
- The `build-prompt` skill is a convenience layer on top of these docs, not a replacement for them.
