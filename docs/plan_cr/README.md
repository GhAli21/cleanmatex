# Plan CR Status

This folder is a historical or secondary planning corpus.

## Current Rule

- `docs/plan/` is the approved planning authority
- use this folder for legacy reference, comparison, or backlog mining only
- when a plan here is still useful, reconcile it into `docs/plan/` instead of treating both copies as equal authorities

## Why This Folder Still Exists

The folder contains many detailed plan drafts that may still be useful during gradual cleanup, but they should not compete with the approved planning area.

## Best Use

1. check whether an equivalent or newer document already exists in `../plan/`
2. copy forward useful material only when it still matches current implementation direction
3. avoid citing this folder first when a `docs/plan/` document already covers the topic

## Reconciliation Aid

Use `../plan/plan_cr_reconciliation_map.md` before interpreting legacy `plan_cr` files. It maps current topic authority and documents PRD-number collisions between the two planning systems.
