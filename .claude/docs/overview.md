## 📋 PROJECT OVERVIEW

**CleanMateX** is a multi-tenant SaaS platform for laundry and dry cleaning management, targeting the **GCC region** with bilingual support (English/Arabic with RTL).

### Core Value Proposition
The platform provides **end-to-end workflow management** from order intake through delivery, with emphasis on:
- ✅ Per-piece tracking
- ✅ Quality assurance gates
- ✅ Digital-first customer engagement
- ✅ Multi-branch support
- ✅ Real-time order tracking

### Target Market
- **Primary**: GCC countries (Saudi Arabia, UAE, Oman, Kuwait, Bahrain, Qatar)
- **Currency**: OMR (Omani Rial) - default
- **Timezone**: Asia/Muscat
- **Languages**: English + Arabic (RTL mandatory)

### Key Differentiators
- 🌐 **Bilingual**: Full English/Arabic support with RTL
- 🏪 **Multi-branch**: Support multiple locations per tenant
- 📱 **Mobile-first**: Customer, driver, and store apps
- 💰 **GCC Payment Gateways**: HyperPay, PayTabs, Stripe
- 📲 **WhatsApp Integration**: Order updates via WhatsApp Business API
- 🔐 **Enterprise Security**: Row-Level Security (RLS) for data isolation

### Business Model
- **SaaS Subscription**: Tiered pricing (Free, Starter, Growth, Pro, Enterprise)
- **Transaction Fees**: Optional per-SMS/WhatsApp charges
- **Marketplace Commission**: Future revenue stream
- **White-label Options**: For enterprise clients

### Development Philosophy
- **Solo Developer Focus**: Built for one-person development with AI assistance
- **Free Tools Priority**: Use open-source and free-tier services
- **Progressive Enhancement**: Start simple, add complexity based on user needs
- **Revenue First**: Focus on features that generate income
