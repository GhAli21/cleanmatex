# CleanMateX i18n Guide

Use folder-based locale catalogs under `web-admin/messages/en/**` and `web-admin/messages/ar/**`.

## Rules

- Search existing keys before adding new ones.
- Reuse `common.*` for generic UI strings.
- Keep the locale file tree and leaf-key set aligned across `en` and `ar`.
- A namespace may be a file or a folder, never both.
- Use `index.json` inside a namespace folder when root keys must stay at that namespace level.
- Do not import locale JSON directly in components.
- Validate with `npm run check:i18n`.

## Example structure

```text
web-admin/messages/
  en/
    common.json
    orders/index.json
    orders/detail.json
  ar/
    common.json
    orders/index.json
    orders/detail.json
```
